# Advent of Code 2018 TypeScript

## Installation

Execute `nix-shell` directly or install `direnv`.

Then: `pnpm install`

## Testing

`pnpm t`

`pnpm t -- --watch`

`pnpm exec jest --watch`

## Building

`pnpm run build`

Build in watch mode:

`pnpm run watch`

## Running

`pnpm start`

## Misc

Remove build artifacts:

`pnpm clean`
